Use keyboard arrows to move the paddle.
Use A or Q to move the piece left, D to move the piece right, S to move the piece down, and Z ou W to rotate the piece.
Use Space to hard drop the piece.
Use F2 to start a new game.
Use P to pause / unpause the game

Some bricks have special effects when broken, try to discover them !
